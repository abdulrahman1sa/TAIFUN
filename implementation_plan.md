# دليل الشعب (University Sections Group Directory) - Implementation Plan

## Overview
A web application to organize Telegram group links for university sections.
Target Audience: University Students.
Language: Arabic (Saudi Dialect).

## Tech Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Database: SQLite (via Prisma)
- Icons: Heroicons
